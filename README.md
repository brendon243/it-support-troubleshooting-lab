# it-support-troubleshooting-lab

This project showcases hands-on simulations of real-world IT support scenarios. Each case demonstrates a structured troubleshooting approach including investigation, diagnosis, resolution, validation, and reflection.

## Project Overview
The goal of this project is to simulate common Tier 1 help desk issues and document how they are diagnosed and resolved in a professional, structured format.

## Case Dashboard
View all documented cases here:  
[Case Dashboard](case-dashboard.md)

## Included Scenarios
- Authentication failures (login issues)
- Network connectivity problems
- Hardware/display issues
- Application errors
- System performance degradation

## Tools & Technologies
- Windows 10
- Task Manager
- Command Prompt (`ipconfig`, `ping`)
- Network Settings
- System Configuration

## Skills Demonstrated
- Troubleshooting methodology
- Root cause analysis
- Technical documentation
- End-user support mindset
- Problem isolation and resolution

## Project Structure
- `cases/` → Individual case logs
- `case-dashboard.md` → Overview of all cases

## Purpose
This project is designed to reflect real-world help desk responsibilities and demonstrate readiness for an entry-level IT support role.
