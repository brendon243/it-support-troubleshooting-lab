## Ticket 005 - System Performance Degradation

## Scenario
The user reports that their system is running very slowly.

## System Context
Windows 10 workstation

## Investigation
- Opened Task Manager to review CPU and memory usage
- Identified multiple high-resource applications
- Reviewed startup programs

## Diagnosis
Excessive background applications were consuming system resources.

## Action Taken
- Ended unnecessary processes
- Disabled non-essential startup programs
- Restarted system

## Result
System performance improved significantly.

## Reflection
Performance issues are commonly caused by resource overload and can often be resolved through system optimization.

## Validation
- Observed reduced CPU and memory usage in Task Manager
- System responsiveness improved after restart
